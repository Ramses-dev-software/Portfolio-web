Most used Technologies
Dart

Flutter
Flutter es un Framework que acompaña al lenguaje de Dart, lo uso mucho en mis proyectos debido a que puede crear distintos tipos de aplicaciones con una sola base de código, lo que me ahorra horas de trabajo para adaptar una aplicación a diferentes dispositivos.
Kotlin
Kotlin es la siguiente evolucion del lenguaje de Java. Lo manejo mucho en el apartado de Scripts, Backend y Servidores web, es muy util si deseas un lenguaje embebido y facil de usar.
JavaScript
JavaScript es mi repertorio de lenguaje para la web. Es muy util en cuestion de darle "Interactividad" a mis paginas hechas en React o con codigo nativo de HTML y CSS.

Tech Stack

Habilidades

Dart

Dart es un lenguaje de programación desarrollado por Google, conocido por su uso en el desarrollo de aplicaciones móviles y web. Se utiliza principalmente con el framework Flutter para construir interfaces de usuario atractivas y dinámicas.

Kotlin

Kotlin es un lenguaje de programación moderno utilizado para desarrollar aplicaciones de Android. Es conocido por su interoperabilidad con Java y su sintaxis concisa, lo que lo hace popular entre los desarrolladores de aplicaciones móviles.

Java

Java es un lenguaje de programación popular utilizado en una variedad de aplicaciones, desde el desarrollo de aplicaciones móviles hasta el desarrollo de software empresarial. Es conocido por su portabilidad y su amplia comunidad de desarrolladores.

Python

Python es un lenguaje de programación versátil utilizado en una amplia gama de aplicaciones, como desarrollo web, análisis de datos, inteligencia artificial y aprendizaje automático. Es conocido por su sintaxis clara y legible.

JavaScript

JavaScript es un lenguaje de programación ampliamente utilizado en el desarrollo web. Se utiliza para agregar interactividad y funcionalidad dinámica a sitios web y aplicaciones web.

HTML5

HTML5 es la última versión del lenguaje de marcado estándar utilizado para crear y estructurar contenido en la web. Se utiliza para definir la estructura de una página web.

CSS3

CSS3 es la última versión de la hoja de estilo en cascada (CSS), que se utiliza para controlar el aspecto y el diseño de una página web. Se utiliza para aplicar estilos como colores, fuentes y diseños a elementos HTML.

Flutter

Flutter es un framework de código abierto desarrollado por Google para crear aplicaciones móviles nativas para iOS y Android desde una base de código única. Es conocido por su rendimiento rápido y su capacidad para crear interfaces de usuario atractivas.

TensorFlow

TensorFlow es una plataforma de código abierto para el aprendizaje automático desarrollada por Google. Se utiliza para construir y entrenar modelos de aprendizaje automático en una variedad de aplicaciones, como reconocimiento de imágenes, procesamiento de lenguaje natural y análisis de datos.

Google AI Platform

Google AI Platform es una plataforma en la nube que permite a los desarrolladores construir, entrenar y desplegar modelos de aprendizaje automático a escala. Ofrece herramientas y servicios para ayudar en todas las etapas del ciclo de vida del desarrollo de modelos de IA.

Google Cloud

Google Cloud es una plataforma en la nube que ofrece una amplia gama de servicios de infraestructura, almacenamiento, bases de datos, inteligencia artificial y más. Se utiliza para construir y escalar aplicaciones en la nube de manera segura y eficiente.

Docker

Docker es una plataforma de código abierto que permite a los desarrolladores empaquetar, distribuir y ejecutar aplicaciones en contenedores. Se utiliza para crear entornos de desarrollo y despliegue consistentes y portátiles.

Git

Git es un sistema de control de versiones distribuido utilizado para rastrear cambios en el código fuente durante el desarrollo de software. Es ampliamente utilizado por equipos de desarrollo para colaborar en proyectos y gestionar el código de manera eficiente.

Github

GitHub es una plataforma de desarrollo colaborativo basada en la web que utiliza Git para controlar versiones de código y alojar proyectos de software. Se utiliza para compartir código, colaborar en proyectos y gestionar el desarrollo de software de manera eficiente.
